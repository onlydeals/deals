Why Your Links Might Not Work

The most common issue with eBay affiliate links is incorrect encoding of the mpre parameter. Here's the correct way to generate Rover links:

function roverLink(rawUrl, customid = "") {
  // Normalize the eBay URL first
  const itemUrl = normalizeEbayItemUrl(rawUrl);

  // Create the base Rover URL
  const u = new URL("https://rover.ebay.com/rover/1/711-53200-19255-0/1");
  u.searchParams.set("campid", CAMPID);
  u.searchParams.set("toolid", "10001");
  if (customid) u.searchParams.set("customid", customid);

  // CRITICAL: Properly encode the mpre parameter
  u.searchParams.set("mpre", itemUrl);

  return u.toString();
}

Make sure you're not double-encoding the URL. The URLSearchParams.set() method will handle encoding automatically.
